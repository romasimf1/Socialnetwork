# ConnectSphere

ConnectSphere is a social networking platform focused on community building and personalized content sharing. This repository contains a minimal project skeleton following the high‑level specification provided in the project brief.

## Project Structure

- `backend/` – Node.js (Express) application.
- `frontend/` – basic React client.

## Features (planned)

- User profiles with privacy settings.
- Personalized news feed and recommendations.
- Groups, events, and messaging.
- Content moderation and analytics.

The current codebase is only a starting point. More services and documentation will be added in future updates.
